To run DNN:
    python3 dnn.py

To run random forest:
    python3 random_forest.py

To LDA/QDA:
    python3 lda_qda.py


To install dependencies:
pip3 install -r requirements.txt

Data can be found at: https://drive.google.com/open?id=11Nap6tKbkpEqMJBc8hHwr8h4Q7m_oNQF

or  ./download.sh

data files are expected to be in a subdirectory /data located in the working directory

Note: Data files may be protected, chmod 644 train.csv, test.csv, weather.csv
